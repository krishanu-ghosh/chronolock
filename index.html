Why ChronoLock?

Designed for the paranoid developer. ChronoLock protects your AWS keys, GitHub tokens, and Seed phrases by forcing a physical "human-in-the-loop" process. Secrets never touch the disk in plaintext and never appear on screen.

Quick Install

user@linux:~$ git clone https://github.com/krishanu-ghosh/chronolock.git user@linux:~$ cd chronolock user@linux:~$ go build -o chronolock cmd/main.go # Setup your vault (encrypts in-place) user@linux:~$ sudo ./chronolock --setup keys.json

The Security Model

  • Immutable Storage: Uses chattr +i to make your vault undeletable by standard users.
  • Time-Based Keys: Passwords rotate every minute based on a mental algorithm.
  • Zero Stdout: Secrets are piped directly to xsel (system clipboard).
